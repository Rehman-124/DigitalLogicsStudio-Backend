# Code of Conduct

## Our Pledge

We are committed to a respectful, inclusive, and professional project environment for contributors, maintainers, users, students, and reviewers.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being clear about technical risks.
- Give actionable feedback.
- Credit others' work.
- Keep discussions focused on the project and user impact.
- Be patient with first-time contributors.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Dismissive comments about skill level, background, identity, or language.
- Publishing private information without permission.
- Repeated disruption of technical discussions.
- Threats, intimidation, or abusive language.

## Enforcement

Maintainers may:

- Ask for clarification or edits.
- Remove comments or content.
- Temporarily restrict participation.
- Permanently ban repeat or severe offenders.

Enforcement should be proportional, documented when appropriate, and focused on keeping the project healthy.

## Reporting

Report conduct concerns privately to the maintainers through repository owner contact channels. Include links, screenshots, and context where possible.

## Scope

This code applies in project repositories, issues, pull requests, discussions, community spaces, and any project-related communication.
